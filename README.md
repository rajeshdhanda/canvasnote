# CanvasNote

Linux note-taking app for 2-in-1 devices with stylus support, palm rejection, and drawing tools.

![CanvasNote Toolbar](images/toolbar.png)

## Installation

```bash
sudo snap install canvasnote
```

OR

[![Get it from the Snap Store](https://snapcraft.io/en/dark/install.svg)](https://snapcraft.io/canvasnote)

## Features

- **Drawing**: Pen, pencil, highlighter, eraser, shapes, colors
- **Text**: Keyboard input with formatting
- **Notes**: Multi-page A4 notes or infinite canvas
- **Templates**: Blank, ruled, grid, dot grid
- **Organization**: Subject-based library with auto-save
- **Export**: PNG and PDF

## Quick Start

1. Create a subject in the sidebar
2. Click "+ Note" and choose A4 Notes or Canvas
3. Select a template
4. Start drawing or typing
